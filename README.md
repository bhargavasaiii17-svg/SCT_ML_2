# Customer Segmentation using K-Means Clustering

## Objective
Segment customers into different groups based on their Annual Income and Spending Score.

## Dataset
Mall Customers Dataset

## Features Used
- Annual Income (k$)
- Spending Score (1-100)

## Algorithm
- K-Means Clustering

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-Learn
- Google Colab

## Results
- Applied Elbow Method to determine optimal clusters
- Identified 5 customer segments
- Visualized customer groups and centroids

## Outcome
Successfully performed customer segmentation using K-Means Clustering and gained hands-on experience in unsupervised machine learning.
